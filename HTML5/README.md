## HTML5
### HTML5 的目标
  - 创建更简单的 Web 应用，写出更简洁的 HTML 代码
    - 提供了大量可以替代脚本的属性标签
    - 提供了大量的语义化结构标签
    - 提供了一个统一的互联网通用标准
  - 放心使用 HTML5 的理由
    - 兼容性：在老版本的浏览器上也可以正常运行
    - 实用性：HTML5 内部并没有封装什么复杂的、不切实际的功能,只是封装了简单实用的功能
    - 非革命性的发展：内部功能不是革命性的，而是在 HTML4上发展
  - HTML5 解决的问题
    - Web 浏览器之间的兼容性很低
    - 文档结构不够明确
    - Web 应用程序的功能受到了限制
